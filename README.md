1. Login Interface
The user opens the DigiLocker application and logs in using mobile number and OTP authentication.

*Interface Elements:*
- Mobile Number Input Field  
- Get OTP Button  
- OTP Verification  
- Login Button  
- Sign Up Option  

*Purpose:*  
To authenticate the user securely before accessing documents.

---

## 🏠 2. Home Dashboard Interface
After successful login, the home dashboard is displayed. It provides access to different features.

*Interface Elements:*
- Issued Documents  
- Uploaded Documents  
- Search Documents  
- My Profile  
- Navigation Icons  

*Purpose:*  
To provide access to all DigiLocker features from a single dashboard.

---

## 📄 3. Issued Documents Section
The user clicks on the “Issued Documents” option to fetch government-issued documents.

*Interface Elements:*
- Document Categories  
- Department List  
- Fetch Document Option  
- Document Cards  

*Purpose:*  
To retrieve verified documents from government databases.

---

## 📥 4. Document Fetch Screen
The user selects a department and enters required details.

*Interface Elements:*
- Select Department Dropdown  
- Document Number Field  
- Full Name Field  
- Fetch Now Button  

*Purpose:*  
To allow users to request and fetch documents digitally.

---

## 📑 5. Document Display Screen
After retrieval, the document is shown on screen.

*Interface Elements:*
- Document Preview  
- User Details  
- Download Button  
- Share Button  

*Purpose:*  
To allow users to view, download, and share documents.

---

## 🔄 Feature Flow
Login → Home Dashboard → Issued Documents → Select Department → Enter Details → Fetch Document → View Document

---

## ⭐ Key Features
- Secure OTP Login  
- Government Verified Documents  
- Digital Document Storage  
- Easy Document Sharing  
- Paperless Access  
- Quick Retrieval System  

---

## ✅ Conclusion
The DigiLocker Issued Document Fetch feature allows users to securely retrieve and store government-issued documents digitally. It reduces the need for physical documents and provides easy access anytime. The system ensures security, reliability, and convenience for users.
